# Installation

## Stable release

To install pykawa, run this command in your terminal:

```sh
uv add pykawa
```

Or if you prefer to use `pip`:

```sh
pip install pykawa
```

## From source

The source files for pykawa can be downloaded from the [Github repo](https://github.com/dangilman/pykawa).

You can either clone the public repository:

```sh
git clone https://github.com/dangilman/pykawa
```

Or download the [tarball](https://github.com/dangilman/pykawa/tarball/main):

```sh
curl -OJL https://github.com/dangilman/pykawa/tarball/main
```

Once you have a copy of the source, you can install it with:

```sh
cd pykawa
uv sync
```
